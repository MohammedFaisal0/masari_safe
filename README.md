# Masari Safe Route

## Project Info

School bus tracking application for parents relative to their children's safe commute.

## How to run locally

You need Node.js & npm installed.

```sh
# Step 1: Install dependencies
npm i

# Step 2: Start the development server
npm run dev
```

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
